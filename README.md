Batch Rename Tool for Unity
A lightweight and efficient Unity Editor extension for renaming multiple objects at once.
Designed to speed up your workflow and keep your hierarchy clean.

Features
Replace text inside object names

Add prefixes and suffixes

Automatic numbering (01, 02, 03…)

Live preview before applying changes

Works with GameObjects, prefabs, and assets

Editor‑only (no runtime code)

Installation
Download the .unitypackage file.

In Unity, go to:
Assets → Import Package → Custom Package…

Select the downloaded file and click Import.

Open the tool from:
Tools → Batch Rename Tool
