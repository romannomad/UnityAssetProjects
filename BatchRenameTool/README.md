Overview
The Batch Rename Tool is a lightweight yet powerful Unity Editor extension that helps creators rename large groups of GameObjects in seconds. Whether you're organizing levels, cleaning up imported assets, or enforcing naming conventions, this tool gives you full control with a clean UI and instant preview.

Perfect for level designers, technical artists, environment artists, programmers, and anyone who wants a tidy hierarchy.

Features
Prefix & Suffix — Add text before or after names

Search & Replace — Replace any substring in selected names

Regex Support — Advanced pattern‑based renaming

Smart Numbering Modes

Global numbering

Restart per parent

Siblings‑only

Children‑only

Custom Name Templates

{name} — final processed name

{parent} — parent object name

{index} — numbering index

{depth} — hierarchy depth

Case Conversion — lowercase, UPPERCASE, Title Case

Sorting Options — hierarchy order, alphabetical, reverse alphabetical

Recursive Renaming — rename children automatically

Indented Live Preview — see exactly how names will look

Undo Support — fully integrated with Unity’s Undo system

How to Use
Select one or more GameObjects in the Hierarchy

Open the tool via:
Tools → Batch Rename Tool

Configure your renaming options

Review the live preview

Click Rename Selected Objects
